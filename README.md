# AI Resume Builder : Create ATS-Friendly Resumes with AI 🚀

### Introduction

AI Resume Builder is a full-stack web application designed to help users create professional, ATS-friendly resumes in minutes.

The platform combines modern resume design with the power of AI, allowing users to generate, enhance, customize, and download resumes through an intuitive and user-friendly interface. By leveraging Gemini AI, users can transform basic resume descriptions into impactful, recruiter-ready content.

---

# The Problem

Creating a professional resume is challenging for many students, freshers, and job seekers.

Common issues include:

* Difficulty writing strong project and experience descriptions.
* Lack of design skills for creating visually appealing resumes.
* Limited knowledge of ATS (Applicant Tracking System) requirements.
* Time-consuming formatting and customization.
* No centralized platform to manage and update resumes efficiently.

As a result, many candidates submit resumes that fail to effectively showcase their skills and achievements.

---

# Our Solution

AI Resume Builder simplifies the entire resume creation process through intelligent automation and customization.

Users can:

* Create and manage resumes through a guided form.
* Choose from professionally designed resume templates.
* Customize template colors and styling.
* Use Gemini AI to enhance descriptions, projects, skills, and achievements.
* Preview changes in real time.
* Download resumes as high-quality PDFs anytime.

The platform enables users to focus on their achievements while AI helps present them professionally.

# Installation / Quickstart

```bash
# Clone Repository
git clone <repository-url>

# Frontend Setup
cd client
npm install

# Backend Setup
cd server
npm install

# Add Environment Variables
MONGODB_URI=
JWT_SECRET=
GEMINI_API_KEY=

# Start Backend
npm run server

# Start Frontend
npm run dev
```
